<h1>About this Application</h1>
<b>IMPORTANT: this fork is currently under heavy construction and is in a non-functional state.</b>

TweetNest is a browsable, searchable and easily customizable archive and backup for your tweets, made in PHP. It runs on a web server. It requires the following:

* PHP 5.2 or higher (or 5.1 with the PECL JSON module installed)
* MySQL 4.1 or higher

<h1>How to Install</h1>
To figure out how to install it, please point your browser to:

<a href="http://pongsocket.com/tweetnest/">http://pongsocket.com/tweetnest/</a>

And go to the "Installation" section.
